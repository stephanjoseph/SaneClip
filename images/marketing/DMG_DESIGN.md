# DMG Installer Design

## Correct Design (v1.1+)

**NO custom background.** Use default Finder background.

The dark custom backgrounds hide the Applications folder icon because macOS folder icons are blue and blend into dark navy backgrounds.

### Settings
- Window size: 540 x 380
- Icon size: 128
- App icon position: 140, 200
- Applications link position: 400, 200
- No background image

### Reference
See `dmg-installer-correct.png` for the approved design.
